# -Construindo-um-Projeto-Logico-de-Banco-de-Dados-do-Zero

 **Resumo do Projeto:** Este projeto apresenta a construção de um banco de dados relacional, desde a modelagem lógica até a implementação de consultas SQL. Focado em um cenário de transações comerciais, é um sistema que estrutura a relação entre clientes, vendedores e compras. 

 **Ferramentas Utilizadas:**

- PostgreSQL (SGBD Relacional)
- dbdiagram.io (Modelagem de Dados)
